# null-safety-cpp

## 概要

C++でのNULL安全モドキのお試し実装です。

## 説明

- 共有ポインタ版
    - `nullable_ptr<T>`はNULL許容共有ポインタです。
        - `*`演算子や`->`演算子で直接リソースにアクセスすることはできません。
        - リソースにアクセスするには、`nonnull_ptr<T>`に変換します。変換する方法は`nonnull_ptr<T>`の項を参照してください。
        - 実装の都合上`shared()`関数で中身の`std::shared_ptr<T>`を取れてしまうため`.shared()->`とやるとNULLチェックを迂回してリソースにアクセスできてしまいますが、しないでください。
    - `nonnull_ptr<T>`は非NULL保証共有ポインタです。
        - 非NULLが保証されているため、`*`や`->`を使って安全にリソースにアクセスできます。
        - デフォルト構築できません。
        - `nullptr`から構築できません。
        - 中身がNULLになり得るようなほとんどのポインタから直接的に構築できません。
        - 構築・取得する方法:
            - `make_nonnull<T>`を呼び出します。
            - `nullable_ptr<T>::if_nonnull()`で、`nullable_ptr<T>`が非NULLだった場合に呼ばれるコールバックで取得します。
            - `nullable_ptr<T>::nonnull_force()`で、`nullable_ptr<T>`から強制的に変換します。
- ユニークポインタ版は未実装です。実装するかは未定です。
- 生ポインタ版は未実装です。実装するかは未定です。
- オプショナル版は未実装です。実装するかは未定です。
